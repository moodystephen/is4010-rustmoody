> Modifying files outside `src/` will break automated grading.

## Setup

1. Install Rust: <https://rust-lang.org/> (follow the installation instructions)
2. Fork this repo on GitHub, then clone your fork:
   ```bash
   git clone https://github.com/YOUR-USERNAME/is4010-rust.git  # replace YOUR-USERNAME
   ```
3. GitHub Actions is automatically enabled in forks. After your first push, visit the **Actions** tab in your fork to see test results.

## Running a lab

```bash
cd path/to/week09                # replace with path to the current week's folder
cargo check                      # fast syntax check (no test run)
cargo test                       # must pass before submitting
cargo fmt                        # format your code
cargo clippy -- -D warnings      # no warnings allowed
git add week09/ && git commit -m "Complete lab 09" && git push
```

`cargo fmt` automatically reformats your code to match Rust's standard style — just run it and the edits happen in place. `cargo clippy` is Rust's built-in linter; the `-- -D warnings` flag used here (and in CI) treats every warning as an error, so your code must be warning-free to pass.

Each week's `labXX.md` has the full task list. A green checkmark on the **Actions** tab means full credit.

## GenAI prompts when you're stuck

Copy your error or code and use one of these patterns:

| Situation | Prompt to use |
|-----------|---------------|
| Compiler error you don't understand | "I'm writing Rust and got this error: [paste error]. Explain what it means and what I should change." |
| Don't know how to start a `todo!()` | "I need to implement this Rust function: [paste signature + doc comment]. Explain the approach before showing any code." |
| Test is failing | "This Rust test fails: [paste test]. My implementation is: [paste fn]. What's wrong?" |
| Borrow checker rejecting your code | "Rust's borrow checker rejects this: [paste code and error]. Explain the rule being violated and how to fix it." |
| Clippy warning you don't understand | "cargo clippy gave me this warning: [paste warning]. What does it mean and how do I fix it?" |
